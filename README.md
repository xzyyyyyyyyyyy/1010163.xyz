## 技术栈与实现
- 单文件静态站（HTML + 内联 CSS/JS）
- 响应式布局（CSS Grid / Flexbox / media queries）
- 深色/浅色主题切换（通过 `data-theme` + localStorage）
- 简单的滚动与可见性动画（IntersectionObserver）
